# btsMapDemo

## Setup

1. Clone the repo
```bash
$ git clone git@github.com:dastan801/btsMapDemo.git
$ cd btsMapDemo/
```
2. Install dependencies from [CocoaPods](http://cocoapods.org/#install)
```bash
$ pod install
```
3. Open the Xcode workspace at `BtsMapDemo.xcworkspace`.
```bash
$ open BtsMapDemo.xcworkspace
```
